## 节标题 {#jbt}

### 小节标题 {#xjbt}

这是《Steem 指南》一书的模板。为了保证将来统稿的质量，减少不必要的麻烦，请各位作者从一开始均以下要求写书稿。

- 本书统一用标准的 markdown 语法来书写。

- 章节标题后面的花括号里，是用来生成网页的网页文件名称，由于书稿网站的网址不支持中文，所以统一用标题拼音的首字母。

- 章标题和节标题之间不要有正文。也就是说，所有的正文文字都属于某个节。

- 本书最多分到三级标题（即“小节”），也就是 markdown 里的三个 # 号。不要出现四级标题。如果需要的话，用粗体表示更低一级的标题即可。

- 插图时请插入本地图片，不要插入网络图片。如果需要插入网络图片，请先把图片下载到本地。同时注意不要侵犯别人图片的版权。

- 选图片时尽量选择静态图片，不要选择 gif 动态图片；当然，如果 gif 效果更好的话，也无妨，但请备好能够替代的静态图片。这是为了方便做 pdf 和纸质书。
